Esqueleto Front End

### Pasos

  • - En la raíz del proyecto ejecutar el siguiente comando npm install

  • - Para ver demanera local el proyecto ejecutar npm run watch que se encuentra en el package.json

  • - Para compilar a dev ejecutar npm run dev

  • - Para compilar a produción npm run prod

  • - Todo es a base de laravel-mix y la version que uso es V5.8 y de node es v14.16.1.

  • - Si desean descomentar alguna carpeta o comentarla puede revisar el .gitignore

  • - Les recomiendo que utilizen nvm

  • - Para el server utilize browsersync,acá les dejo las opcioneshttps://browsersync.io/docs/options

  • - Si desean agregar alguna carpeta, por ejemplos "frontend/documentos", deben agregarlo en el siguiente archivo webpack.mix.js
